# **App Name**: Víctor Sepúlveda (Sekain)

## Core Features:

- Hero Section: Displays name, main phrase, and social media buttons (GitHub, LinkedIn, Threads, contact).
- About Me Section: Presents a brief personal story, inspiring quote, and professional description.
- Skills Section: Showcases technical skills with chips or icons of technologies: Python, FastAPI, Flask, Angular, Ionic, Vue, Firebase, Arduino, MySQL, GitHub.
- Projects Section: Displays project cards with name, brief description, and main technologies.
- Services Section: Lists offered services (Web/Mobile Development, IoT, APIs, Consulting, Support) with icons.
- Contact Form: Enables visitors to send messages (name, email, message) and links to social media.
- Theme Toggle: Allows users to switch between light and dark modes for optimal viewing experience.
- Smooth Scrolling: Implements smooth scrolling navigation to internal section IDs (#inicio, #sobre-mi, etc.).

## Style Guidelines:

- Primary color: Electric blue (#2563EB) to reflect technology and energy.
- Accent color: Light cyan (#38BDF8) to add a vibrant highlight.
- Light background: #F3F4F6 for a clean, modern look in light mode.
- Dark background: #0F172A for an elegant, tech-focused feel in dark mode.
- Font: 'Inter' (sans-serif) for body and headlines to provide a contemporary, neutral style. Note: currently only Google Fonts are supported.
- Rounded borders: 16px to soften the design.
- Generous vertical padding: ~120px between sections for clear separation.
- Subtle hover effects and transitions for a polished user experience.
